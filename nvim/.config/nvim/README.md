## Plugins

| **Category**       | **Plugin**                                                                 | **Description**                                                                     |
| ------------------ | -------------------------------------------------------------------------- | ----------------------------------------------------------------------------------- |
| **Plugin Manager** | [Lazy](https://github.com/folke/lazy.nvim)                                 | Fast and efficient plugin manager                                                   |
| **File Explorer**  | [Neo Tree](https://github.com/nvim-neo-tree/neo-tree.nvim)                 | A modern file explorer that manages the file system and other tree-like structures  |
| **Treesitter**     | [Nvim Treesitter](https://github.com/nvim-treesitter/nvim-treesitter)      | Enhanced syntax highlighting and parsing                                            |
| **Editing**        | [autopairs](https://github.com/windwp/nvim-autopairs)                      | Automatically completes pairs of characters (e.g., parentheses, brackets, quotes)   |
| **UI**             | [lualine](https://github.com/nvim-lualine/lualine.nvim)                    | A fast, customizable statusline                                                     |
|                    | [Tokyo Night](https://github.com/folke/tokyonight.nvim)                    | A clean, dark, and beautiful colorscheme                                            |
|                    | [indent-blankline](https://github.com/lukas-reineke/indent-blankline.nvim) | Provides visible indent guides for better code readability                          |
| **LSP**            | [nvim-lspconfig](https://github.com/neovim/nvim-lspconfig)                 | Quickstart configurations for Neovim's LSP client                                   |
|                    | [Mason](https://github.com/williamboman/mason.nvim)                        | Easily install and manage LSP servers, DAP servers, linters, and formatters         |
|                    | [none-ls](https://github.com/nvimtools/none-ls.nvim)                       | Inject LSP diagnostics, code actions, and more                                      |
| **Completion**     | [nvim-cmp](https://github.com/hrsh7th/nvim-cmp)                            | A highly extensible and customizable completion plugin                              |
|                    | [LuaSnip](https://github.com/L3MON4D3/LuaSnip)                             | A snippet engine for Neovim written in Lua, integrates with `nvim-cmp`.             |
| **Miscellaneous**  | [mini-nvim](https://github.com/echasnovski/mini.nvim)                      | A collection of 40+ Lua modules to enhance Neovim’s functionality.                  |
|                    | [snacks](https://github.com/folke/snacks.nvim)                             | A collection of Quality of Life plugins for Neovim.                                 |

## Info

To disable any plugin just add `enabled = false` in plugin config file
